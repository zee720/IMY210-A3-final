Reflection:

This assignment taught me a lot about using Docker and Strapi. At first it was difficult because several parts of the setup did not work as I expected. My Strapi project kept refreshing when I tried to log in, and I also had problems with Docker ports already being used. It took time to figure out what was wrong and how to fix it.

Even though there were challenges, I learned how to build and run Docker containers, how Strapi works as a backend, and how it connects to a Nuxt frontend. I also learned more about using Git and GitHub for version control.
